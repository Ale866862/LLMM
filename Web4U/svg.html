Animación simple (transform)

Usamos una animación CSS aplicada a un grupo (<g>) dentro del SVG para rotación continua.

<svg width="260" height="120" class="anim-svg">
  <g class="spin" transform-origin="130px 60px">
    <circle cx="130" cy="60" r="36" fill="#b89dff" />
    <rect x="120" y="36" width="20" height="12" fill="#6a3fb5" rx="3" />
  </g>
</svg>

/* styles.css: .anim-svg .spin { animation: spin 8s linear infinite; } */

Referencia: Animaciones SVG
