HTML — etiquetas y uso

Aquí se explican varias etiquetas HTML con ejemplos reales, casos de uso, código para copiar y una imagen SVG que muestra el resultado.

Párrafo — <p>

Uso: agrupar texto en bloques semánticos. Evita usar <br> para maquetación.

Código

<!-- Párrafo básico -->
<p>Este es un párrafo de ejemplo. Utiliza etiquetas semánticas para mejorar accesibilidad.</p>

Resultado

Referencia W3C/WHATWG: Elemento <p>

Enlace — <a>

Uso: navegación entre páginas o enlaces externos. Añade title para información extra.

Código

<a href="https://www.w3.org/" target="_blank" rel="noopener noreferrer" title="Abrir W3C en una pestaña nueva">W3C — especificaciones web</a>

Resultado

Referencia: Elemento <a>

Imagen — <img>

Uso: incrustar imágenes. Siempre proporciona alt descriptivo.

Código

<img src="" alt="Miniatura HTML con tarjeta morada" width="300">

Resultado

Referencia: Elemento <img>

Tabla — <table>

Uso: datos tabulares. No usar tablas para maquetación.

Código

<table>
  <caption>Ventas mensuales</caption>
  <thead>
    <tr><th scope="col">Mes</th><th scope="col">Ventas</th></tr>
  </thead>
  <tbody>
    <tr><td>Enero</td><td>100</td></tr>
  </tbody>
</table>

Resultado

Referencia: Tablas (HTML)

Formulario — <form>

Uso: recopilar datos. Etiqueta inputs y utiliza tipos adecuados.

Código

<form action="#" method="post">
  <label for="nombre">Nombre</label>
  <input id="nombre" name="nombre" type="text" required placeholder="Tu nombre">
  <label for="email">Email</label>
  <input id="email" name="email" type="email" required placeholder="tu@ejemplo.com">
  <button type="submit">Enviar</button>
</form>
