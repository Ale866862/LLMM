Párrafo alineado a la izquierda

Este párrafo tiene texto en negrita y en cursiva. El fondo es azul claro y los colores del texto son diferentes para cada formato.

Párrafo centrado

Este texto está centrado. Aquí se puede usar subrayado y texto en cursiva para resaltar información importante.

Párrafo alineado a la derecha

Este párrafo está alineado a la derecha y muestra cómo los colores cambian según el tipo de texto. Texto tachado puede usarse para mostrar correcciones.

Párrafo justificado

Este texto está justificado. Puedes ver cómo los márgenes son uniformes en ambos lados. Combinando colores, estilos y resaltados, se logra una presentación más atractiva y fácil de leer.
