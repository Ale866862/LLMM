Alineación de Párrafos

Párrafo alineado a la izquierda

Este párrafo está alineado a la izquierda. Es el tipo de alineación predeterminada en la mayoría de los navegadores y se utiliza con frecuencia para el texto principal.

Párrafo centrado

Este párrafo está centrado en la página. Se suele usar para títulos, citas o textos que se quieren destacar.

Párrafo alineado a la derecha

Este párrafo está alineado a la derecha. No es común en texto general, pero puede ser útil para firmar documentos o mostrar fechas.

Párrafo justificado

Este párrafo está justificado. La alineación justificada ajusta los espacios entre las palabras para que los bordes izquierdo y derecho queden alineados. Se usa mucho en textos impresos para dar un aspecto más formal y ordenado.
